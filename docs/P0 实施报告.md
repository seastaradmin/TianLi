### ✅ 3. 后端 API 接口

**文件:** `backend/api_routes.py`

**接口列表:**

#### 交付物接口
```python
GET /api/deliverables           # 获取交付物列表
GET /api/deliverables/{id}      # 获取交付物详情
GET /api/deliverables/{id}/download  # 下载交付物
```

#### 日志接口
```python
GET /api/logs                   # 获取日志 (支持筛选)
GET /api/logs/stream            # 实时日志流 (SSE)
```

#### 指标接口
```python
GET /api/metrics                # 获取系统指标
GET /api/metrics/hero-performance  # Hero 性能统计
```

#### 数据库接口
```python
GET /api/database/status        # 数据库连接状态
GET /api/database/feedback-stats   # 反馈统计
```
